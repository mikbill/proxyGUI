Introduction

Документация API для работы с ProxyGUI.

В правой части можно увидеть примеры API запросов на разных языках (переключение языка доступно в верхней части)

Base URL

https://proxy.nekkoy.pp.ua

Authenticating requests

This API is authenticated by sending an Authorization header with the value "Bearer {YOUR_AUTH_KEY}".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

You can retrieve your token by visiting your dashboard and clicking Generate API token.

Auth

Получение JWT токена для дальнейшей работы с API

POST api/v1/auth

Example request:
curl --request POST \
    "https://proxy.nekkoy.pp.ua/api/v1/auth" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"admin\",
    \"password\": \"hic\"
}"

const url = new URL(
    "https://proxy.nekkoy.pp.ua/api/v1/auth"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "admin",
    "password": "hic"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
